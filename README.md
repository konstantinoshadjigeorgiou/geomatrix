# Geomatrix Theme

## Requirements
- Apache, PHP, MySQL (check mod_rewrite apache module to be enabled)
- Git
- nodejs
- npm
- gulp installed globally

### Setup

1. Import DB

2. Edit .env file -> set local DB settings

3. Navigate to your theme directory (wp-content/themes/geomatrix) and then run:
  - `npm install`
  - `gulp `
